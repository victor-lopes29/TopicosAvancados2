# ToDo API
Exemplo de REST API gerada com o Spring Boot.

###
Acessar os recursos.

* URL padrão do projeto: http://localhost:8080
* URL da o JSON gerado para o Swagger: http://localhost:8080/api-docs
* URL da documentação da API gerada pelo Swagger:  http://localhost:8080/index.html ou  http://localhost:8080
